## Dmit 更换 IP 政策

以下是 Dmit 对更换 IP 的具体规则：

1. **被中国大陆 TCP 阻断** 的 IP 每隔 35 天可以更换一次。
2. 更换 IP 的前提条件：**服务非月付套餐且剩余使用天数 > 35 天**。
3. **被 DDoS 攻击** 的 IP 在 3 天内不可更换。
4. **完全阻断（所有协议全封）** 的 IP 不支持更换。
